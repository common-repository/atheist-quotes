This plugin puts some great quotes by Atheist on the dash of your Wordpress admin panel.

== Description ==

Get some great quotes evcerytime you visit your admin dash on your Wordpres site.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload 'atheist-quotes' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Be sure to backup your adicon “images” folder before you update.

Please visit http://since83.com/files/file/11-atheist-quotes/ for usage instructions.
